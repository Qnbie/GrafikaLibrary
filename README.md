# GrafikaLibrary

BME VIK Számítógépes grafikához készült segéd könyvtár.

## Tartalom
 -  vec4
 -  dnum
 -  Lagrange interpoláció
 -  Catmul-Rom spline
 -  O spline
 -  Besier aproximáció
 -  Hermit interpoláció

## Cél
Egy demo program létrehozása ami a 2020-as őszi félév első házifeladatában szereplő földrészek koordinátáit számolja ki.
